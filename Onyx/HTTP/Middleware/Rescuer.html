    module Onyx::HTTP::Middleware::Rescuer(T)

    Overview

    Rescues T. Firstly calls #handle to handle the error (e.g. log it). Calls #before_next_handler and then the #next_handler.

    See Rescuer::Standard and Rescuer::Silent.
