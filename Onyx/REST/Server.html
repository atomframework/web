    class Onyx::REST::Server

    • Onyx::REST::Server
    • HTTP::Server
    • Reference
    • Object

    Overview

    The Onyx REST server. Basically it is just a wrapper around default HTTP::Server, which logs server start and stop events.

    server = Onyx::REST::Server.new(handlers)
server.bind_tcp("0.0.0.0", 5000)
server.listen

#   INFO -- : ⬛ Onyx::REST::Server is listening at http://0.0.0.0:5000
^C
#   INFO -- : ⬛ Onyx::REST::Server is shutting down!
