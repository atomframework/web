    class Onyx::REST::Renderers::JSON

    • Onyx::REST::Renderers::JSON
    • Reference
    • Object

    Overview

    A JSON renderer. If ::HTTP::Server::Response#error is present, prints it as a JSON object, otherwise renders ::HTTP::Server::Response#view, calling View#to_json on it. It updates the "Content-Type" header only if error of view is present. Should be put after router. Calls the next handler if it's present.
