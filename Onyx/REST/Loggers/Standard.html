    class Onyx::REST::Loggers::Standard

    • Onyx::REST::Loggers::Standard
    • Reference
    • Object

    Overview

    Logs requests colorfully into specified standard logger. It also logs the HTTP::Request#id if it's present.

    logger = Onyx::REST::Loggers::Standard.new

#   INFO -- :     GET /users 200 102μs
#   INFO -- :     GET /favicon.ico 404 52μs
#   INFO -- :    POST /users 201 3.74ms
