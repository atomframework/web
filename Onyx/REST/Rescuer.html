    abstract class Onyx::REST::Rescuer(T)

    • Onyx::REST::Rescuer(T)
    • Reference
    • Object

    Overview

    Rescues T. Firstly calls #process to handle the error (e.g. log it). Then if #handler is present, calls #before_handler and then the handler itself. Otherwise calls #fallback.

    See Onyx::REST::Rescuers::Standard.
